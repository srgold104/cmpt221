Troubleshooting
===============
This section describes some common issues that can arise and possible solutions.

Issue #1
--------
A common issue encountered during lab was a failure to ensure you are using the correct branch.

The best solution for this problem is to read and execute the lab instructions carefully and in order.

Issue #2
--------
A common issue encountered in lab is a failire to activate the virtual environment.

The best solution for this problem is to read and execute the lab instructions carefully and in order.


Issue #3
--------
A very common issue encountered in lab is difficulty pulling from and pushing to Git Hub.

The best solution for this problem is to read and execute the lab instructions carefully and in order. And using Google...a lot.
